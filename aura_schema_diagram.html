Base Record

Acts as the foundational metadata anchor connecting direct Supabase Auth session identifiers with specialized downstream financial entities.

id uuid PK
auth_user_id uuid FK, UK
display_name varchar
state varchar

user_roles

Access Security

Maintains the primary authorization mapping for Aura users. Fully hardened from client tampering; policy validation occurs at token level.

id uuid PK
auth_user_id uuid UK
role varchar
created_at timestamptz

user_identity

PII Shield

Maintains personally identifiable information (first and last names, phones, raw emails) that are obfuscated directly inside client compilation layers.

id uuid PK
auth_user_id uuid UK
first_name text obfuscated
email text UK

financial_twins

Analysis Engine

Holds computed financial metrics, progress tracking variables, and aggregate summaries for the customer's digital projection.

id uuid PK
profile_id uuid FK, UK
net_worth numeric
plan_health text

assets / liabilities

Ledger items

Multi-record tables storing distinct entries for physical portfolio nodes (brokerage, real estate, cash) or debt liabilities (student loans, car logs).

profile_id uuid FK
asset_type / ... varchar
current_value numeric
growth_rate numeric

state_assumptions

System Static

Houses tax metrics, appreciation constants, and cost-of-living adjustments for California, Texas, the United States, etc., mapped by states.

id uuid PK
state_code varchar UK
effective_tax_rate numeric
property_tax_rate numeric
